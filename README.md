# FormAdapter

Automatic schema forms for any UI kit.

This repository is a Bun monorepo for the `@formadapter/*` package family.

## Packages

- `@formadapter/react`

## Development

```sh
bun install
bun run typecheck
bun run test
bun run build
```

## Release

Packages start at `0.0.0`. Future published versions should be created with changesets and released by GitHub Actions.
